Local storage is the set of data that is permanently stored in the local store in the browser. The data will remain in the browser until the data is deleted by manually or using Javascript. The data is set as key and value. A maximum of 5 to 10 MB of data can be stored in local storage.

Session storage is the set of data in a browser session and the data is deleted when the browser closes or the page session ends. Here also the data is set as key and value. Session storage can store a maximum of 5 MB of data.
